Agent Forge

QualityGate

Decides whether an agent passes or fails based on its ScoreReport, configurable thresholds, and coverage checks.

Interface

import QualityGate from "agent-forge/quality_gate";

const gate = new QualityGate(0.85);  // default threshold

const result = gate.check(scoreReport);
// result.passed: boolean
// result.warnings: string array
// result.blocked: boolean (coverage gaps)
// result.compositeScore: number

QualityGateResult

interface QualityGateResult {
  passed: boolean;       // composite greater-or-equal threshold AND not blocked
  warnings: string[];    // non-blocking alerts
  blocked: boolean;       // true if coverage gaps exist
  compositeScore: number;
}

Pass Conditions

An agent passes if and only if:

  1. composite score greater-or-equal threshold
  2. coverage gaps length equals zero

If coverage gaps exist, blocked is true regardless of score.

Warning Conditions

Warnings are attached (but don't block) for:

  • testPassRate less than 0.5 — "Low test pass rate"
  • judgeScore less than 0.5 — "Low LLM judge score"
  • generalizationWarning present — shadow eval failure
  • coverageGaps.length greater than 0 — listed as warning AND blocks

Default Threshold

composite = 0.6 * testPassRate + 0.4 * judgeScore

// With default 0.85 threshold:
// - 100% test pass + 100% judge = 1.0 = PASS
// - 100% test pass + 50% judge = 0.8 = FAIL
// - 50% test pass + 100% judge = 0.7 = FAIL
// - 85% test pass + 85% judge = 0.85 = PASS

Customization

// Different threshold per task
const strictGate = new QualityGate(0.95);
const lenientGate = new QualityGate(0.70);

// Change threshold dynamically
gate.setThreshold(0.90);
