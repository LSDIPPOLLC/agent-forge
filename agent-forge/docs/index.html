Agent Forge

A meta-framework for creating and auto-refining LLM-driven agents

Get Started How It Works

LLM-Driven Generation

Describe your agent in natural language. Agent Forge generates a complete, self-contained TypeScript agent ready to run.

Automated Evaluation

Every iteration is tested against LLM-generated test cases with an LLM judge scoring semantic quality — no manual test writing.

Self-Refining Loop

Failed evaluations feed back into an LLM refiner that rewrites only the broken parts — targeted fixes, not full rewrites.

Overfitting Safeguards

Diversity-enforced test generation, intent tracing, and generalization holdouts ensure agents serve real use cases, not just test suites.

Multi-Agent Execution

Agents can call sub-agents with depth-limited nested execution and cycle detection for compositional workflows.

Pluggable Output

Export finished agents to Claude Skills format (~/.claude/skills/) or as standalone packages.

One Command

agent-forge create "review PRs for logic errors" \
  --tools gh_api,read_file \
  --output claude-skills

Agent Forge generates the agent, runs the eval loop, refines until quality threshold is met, and exports to your target.
