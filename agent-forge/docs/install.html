Agent Forge

Installation

Prerequisites

  • Bun (v1.1+) — Install Bun
  • LLM Backend — Ollama (local) or OpenAI/Anthropic API key

Install Agent Forge

git clone https://github.com/yourusername/agent-forge.git
cd agent-forge
bun install

Configure LLM Backend

Ollama (Local, Default)

# Install Ollama from https://ollama.com
ollama pull llama3.2

# Verify
curl http://localhost:11434

Set base URL if different:

export OLLAMA_BASE_URL=http://localhost:11434
export OLLAMA_MODEL=llama3.2

OpenAI / Anthropic

export OPENAI_API_KEY=sk-...
export OPENAI_MODEL=gpt-4o

Or use Anthropic:

export ANTHROPIC_API_KEY=sk-ant-...
export OPENAI_MODEL=claude-3-5-sonnet-latest
export OPENAI_BASE_URL=https://api.anthropic.com/v1

CLI Installation (Optional)

# Make CLI globally available
bun link

# Now use anywhere
agent-forge --help

Verify Installation

bun run src/cli.ts help
agent-forge — Meta-framework for creating and auto-refining agents

Usage:
  agent-forge create <description>    Create and refine a new agent
  agent-forge list                    List all agents
  agent-forge history <name>         Show version history
  agent-forge inspect <name> [--v N]  Inspect agent code
  agent-forge help                   Show this help

Project Structure

agent-forge/
├── src/
│   ├── agent_schema.ts         # Agent file format
│   ├── generator.ts             # NL → agent code
│   ├── test_generator.ts        # Auto-generate test cases
│   ├── evaluator.ts             # Test runner + LLM judge
│   ├── refiner.ts              # Rewrite from failure report
│   ├── registry.ts              # SQLite persistence
│   ├── quality_gate.ts          # Pass/fail threshold
│   ├── loop_controller.ts       # Orchestration state machine
│   ├── runtime/
│   │   ├── executor.ts         # Single-agent execution
│   │   └── nested_executor.ts  # Multi-agent call graph
│   ├── llm_backend/            # Ollama / OpenAI adapters
│   └── output_adapters/        # Claude Skills / Standalone
└── docs/                        # This documentation site
