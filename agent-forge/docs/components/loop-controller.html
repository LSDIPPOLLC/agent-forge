Agent Forge

LoopController

The orchestration engine that runs the full generate evaluate refine loop until the agent passes or max iterations are reached.

Interface

import { LoopController } from "agent-forge/loop_controller";
import { createOllamaBackend } from "agent-forge/llm_backend/ollama";

const llm = createOllamaBackend();
const controller = new LoopController(llm);

const result = await controller.run({
  description: "review PRs for logic errors",
  name: "pr_reviewer",
  tools: ["gh_api", "read_file"],
  threshold: 0.85,
  maxIterations: 5,
});

// result.state: "DONE" or "FAILED"
// result.passed: boolean
// result.finalScore: number or undefined
// result.bestScore: number or undefined
// result.iterations: number
// result.agentId: string or undefined
// result.versionId: string or undefined

State Machine

IDLE
  (run() called)
 GENERATING
  (schema generated)
 EVALUATING
  (eval complete)
 REFINING
  (refinement done, back to EVALUATING)
  ... loop until:
    DONE (passed gate)
    FAILED (max iterations)

Loop Algorithm

for (iter = 1; iter less-or-equal maxIterations; iter++) {
  if (iter equals 1 and not currentCode) {
    // First iteration: generate
    currentCode = Generator.make(description)
  } else if (currentCode) {
    // Subsequent: evaluate before refining
    TestCases = TestGenerator.generate(description)
    Report = Evaluator.run(agentCode, TestCases)
    Coverage = TestGenerator.checkCoverage(TestCases)
    Report.coverageGaps = Coverage.gaps

    if (QualityGate.check(Report).passed) {
      return { state: DONE, passed: true, finalScore: Report.composite };
    }

    // Refine and loop
    currentCode = Refiner.refine(agentCode, Report)
  }
}
return { state: FAILED, passed: false, bestScore }

Best Score Tracking

The highest-scoring version is tracked across all iterations. If the loop exits without passing, the best version's code is still available in the registry for inspection.

Integration with Registry

  • Agent created (or looked up if exists) on first iteration
  • Every iteration creates a new agent_versions row
  • Every iteration creates an eval_runs row with full failure report
