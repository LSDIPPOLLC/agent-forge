Agent Forge

Registry

SQLite-backed persistence layer for agents, versions, evaluation history, and test cases.

Database Location

# Default
.agent-forge/registry.db

# Custom
AGENT_FORGE_DATA=/path/to/data agent-forge create "..."

Schema

agents
  id          TEXT PRIMARY KEY
  name        TEXT UNIQUE NOT NULL
  description TEXT
  created_at  DATETIME DEFAULT CURRENT_TIMESTAMP
  latest_version INTEGER DEFAULT 0

agent_versions
  id          TEXT PRIMARY KEY
  agent_id    TEXT REFERENCES agents(id) ON DELETE CASCADE
  version     INTEGER NOT NULL
  code        TEXT NOT NULL
  created_at  DATETIME DEFAULT CURRENT_TIMESTAMP
  UNIQUE(agent_id, version)

eval_runs
  id                  TEXT PRIMARY KEY
  agent_version_id    TEXT REFERENCES agent_versions(id) ON DELETE CASCADE
  iteration           INTEGER NOT NULL
  test_pass_rate      REAL
  judge_score         REAL
  composite_score     REAL
  passed              INTEGER
  failure_report      TEXT  -- JSON
  created_at          DATETIME DEFAULT CURRENT_TIMESTAMP

test_cases
  id              TEXT PRIMARY KEY
  agent_id        TEXT REFERENCES agents(id) ON DELETE CASCADE
  input_json      TEXT NOT NULL
  expected_schema TEXT NOT NULL
  traced_to       TEXT  -- JSON array
  diversity_score REAL
  created_at      DATETIME DEFAULT CURRENT_TIMESTAMP

Key Functions

// Agents

getAllAgents(): AgentRow[]

getAgentVersion(agentId: string, version: number): VersionRow | null
getLatestAgentVersion(agentId: string): VersionRow | null
getAllAgentVersions(agentId: string): VersionRow[]

// Evals
createEvalRun(params: EvalParams): EvalRunRow
getEvalRunsForVersion(agentVersionId: string): EvalRunRow[]
getBestEvalForAgent(agentId: string): EvalRunRow | null

// Test Cases
createTestCase(params: TestCaseParams): TestCaseRow
getTestCasesForAgent(agentId: string): TestCaseRow[]
deleteTestCasesForAgent(agentId: string): void

aiosqlite Notes

The registry uses bun:sqlite (Bun's native SQLite bindings). WAL mode is enabled for concurrent read access. Foreign keys are enforced via CASCADE deletes.
