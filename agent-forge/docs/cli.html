Agent Forge

CLI Reference

Commands

create

Create and refine a new agent from a natural language description.

agent-forge create "description" [options]

Options:
  --tools <list>       Comma-separated tools (default: read_file, run_command)
  --output <target>     Output target: claude-skills or standalone (default: claude-skills)
  --threshold <0-1>   Pass threshold (default: 0.85)
  --max-iterations <N> Max refinement iterations (default: 5)

Example:

agent-forge create "review PRs for logic errors" \
  --tools gh_api,read_file \
  --output claude-skills \
  --threshold 0.85 \
  --max-iterations 5

list

List all agents in the registry with their latest status.

agent-forge list
 Agents (3):

  [PASS] pr_reviewer (v4) — Reviews PRs for logic errors
  [FAIL] code_summarizer (v5) — Summarizes code files
  [NEW]  test_generator (v0) — Generates test cases

history

Show version history and scores for a specific agent.

agent-forge history <name>
 pr_reviewer — 4 version(s):

  v1: score=0.42 pass=no
  v2: score=0.58 pass=no
  v3: score=0.71 pass=no
  v4: score=0.89 pass=yes

inspect

Inspect the code of a specific agent version.

agent-forge inspect <name> [--v N]
# Latest version
agent-forge inspect pr_reviewer

# Specific version
agent-forge inspect pr_reviewer --v 3

eval

Re-run evaluation against the latest version of an agent.

agent-forge eval <name>

refine

Force a refinement loop starting from the current version.

agent-forge refine <name>

export

Export the latest passed version to a directory.

agent-forge export <name> --output ./my-agents/
